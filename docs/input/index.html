Input Handling

In this section we'll learn how to handle input from mouse and keyboard. You'll see that this isn't really complex and in fact it is quite similar to handlers in the Java Swing library. Input handling should be really easy and intuitive to allows the developer to concentrate on other features.

Let's start saying that some classes in the Engine extends an Input "module", that allows them to receive input by the user. These classes are the actual Engine, to handle generic input, and PhysicsObject, to handle input on them. This Input module allows you to add some listeners
