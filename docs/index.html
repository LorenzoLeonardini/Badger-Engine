Getting Started

Let's see how to get start using this Engine. After you have included the library into your Java project and you have created the main class, let's move into the main method.

The following code shows how to create a window and initialize the Engine. 


import org.lorenzoleonardini.badger.Engine2D;

public class HelloWorld
{
	public static void main(String[] args)
	{
		// Create and initialize the 2D engine
		Engine2D engine = new Engine2D();

		// Create the window. Parameters are width, height, scale and title
		engine.createWindow(900, 516, 1, "Hello World!");

		// Start the engine
		engine.start();
	}
}

As you can notice the window is not closing. In fact, differently from GLFW, this Engine assumes that your window must survive until you press the close button. Another thing that you can easily notice is that every second both FPS and Objects counters are displayed in console. We'll give a deeper look to the second one in the future. For now just note that you can turn off these updates by changing this two boolean values: 


engine.logFPS = false;
engine.logObjects = false;
